PROJECT PROCEDURE

해당 작업물은 React를 사용하여 제작된 가구 쇼핑몰 홈페이지입니다.
배치는 Bootstrap을 사용하여 틀을 잡았고 그 외에 슬라이드 기능은 Swiper를
사용하여 기능을 넣었습니다. 상품 관련 장바구니는 Redux를 이용해 제품을
가져올 수 있도록 만들었고 상세 페이지는 React Router로 각각의 페이지를 만들었습니다.
처음 하는 React 작업이다 보니 Swiper 에서부터 작업에 문제가 생겼습니다. 그로 인해
구글에서 검색하여 React Swiper 사용하는 법을 찾아 여러 차례 코드 수정을 하여 성공했고,
Redux를 활용하여 상품을 장바구니로 옮기는 가정에서도 많은 문제가 생겼습니다.
이 부분은 검색해도 어려운 부분이 너무 많아 선생님에게 도움을 받아 해결했습니다.
이것 말고도 이 페이지에 제작은 많은 어려움을 겪었습니다. 끊임없는 오류가 발생했고
해결하기 위해 검색 후 다시 코드를 짜고를 무수히 반복했습니다. Js의 이해심이
부족하여 제가 원하는 만큼의 작업물이 안 나왔고 이번 제작을 기회로 React 경험과
Js의 공부 중요성을 다시 한번 배웠습니다.
